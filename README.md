# WorkFlow Pro - ISO Management System

Hệ thống quản lý vận hành và tài liệu ISO thông minh tích hợp Gemini AI.

## Tài liệu dự án
Toàn bộ thông tin chi tiết về ứng dụng, kiến trúc kỹ thuật và hướng dẫn sử dụng được lưu trữ trong thư mục `/docs`:

1.  [**Tổng quan dự án**](/docs/OVERVIEW.md): Giới thiệu, tầm nhìn và các module cốt lõi.
2.  [**Quản lý ISO**](/docs/ISO_MANAGEMENT.md): Quy trình workflow, soft delete và các tiêu chuẩn hỗ trợ.
3.  [**Kiến trúc Kỹ thuật**](/docs/TECHNICAL_ARCHITECTURE.md): Tech stack, Supabase, AI Engine và phân quyền.
4.  [**Hướng dẫn sử dụng**](/docs/USER_GUIDE.md): Cách vận hành Dashboard, Quản lý ISO và Trợ lý AI.
5.  [**Kế hoạch cải tiến**](/docs/FUTURE_ROADMAP.md): Lộ trình phát triển và các tính năng nâng cao.

## Công nghệ chính
*   **Frontend:** React, TypeScript, Tailwind CSS, Framer Motion.
*   **Backend:** Supabase (Database, Storage, Auth).
*   **AI:** Google Gemini AI.
